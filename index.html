操作符

  • 一元操作符（只能操作一个值的操作符）
    • i++
    • ++i
    • i–
    • –i

      递增和递减操作是在包含它们的语句被求值之后才执行。

    • +
    • -

  • 位操作符

    有符号的整数，第三十二位表示数值的正负
    0表示正数，1表示负数
    正数为纯二进制格式存储
    负数要进行三步（补码=反码+1）

    - 绝对值的二进制码
- 反码
- +1
    • NOT(按位非)
      • ~ 返回数值的反码
    • AND（按位与）
      • & （逻辑乘） 两个数值都为1的时候才返回1
    • OR（按位或）
      • | （逻辑加） 两个数值都为0的时候才返回0
    • XOR
      • ^ 两个数值只有一个1时才返回1，否则返回0
    • 左移
      • <<
    • 有符号的右移

      • >

    • 无符号的右移

      • >

      • 正数的无符号右移和正数的有符号右移结果相同
      • 负数的无符号进行补码形式表示
  • 布尔操作符
    • 逻辑非！
    • 逻辑与&&（*）
    • 逻辑或||（+）
  • 乘性操作符
    • 乘法 *
    • 除法 /
    • 求模 %
  • 加性操作符
    • +
    • -
  • 关系操作符

    • < <= >=

  • 相等操作符
    • ==
    • !=
    • ===

  • 条件操作符

  • 赋值操作符

    • =

Event Handlers

  • 触发

    • 一般：
      • KeyDown事件：有焦点的情况下按下键时触发
      • KeyPress事件：有焦点的情况下按下键时触发
      • KeyUp事件：有焦点的情况下释放键时触发
    • 中英文：
      • 中文：keypress 失效
      • 英文：keydown - > keypress - > keyup

  • 属性

    • charCode：用户按下的字符键对应的ASCII编码
    • keyCode：用户按下的键对应的ASCII编码
