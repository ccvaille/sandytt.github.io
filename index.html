发表于 2016-04-16   |   分类于 前端

Event Handlers

  • 触发

    • 一般：
      • KeyDown事件：有焦点的情况下按下键时触发
      • KeyPress事件：有焦点的情况下按下键时触发
      • KeyUp事件：有焦点的情况下释放键时触发
    • 中英文：
      • 中文：keypress 失效
      • 英文：keydown - > keypress - > keyup

  • 属性

    • charCode：用户按下的字符键对应的ASCII编码
    • keyCode：用户按下的键对应的ASCII编码
