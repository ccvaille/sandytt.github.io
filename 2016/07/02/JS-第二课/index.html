JS 第二课

发表于 2016-07-02   |   分类于 前端

变量

  • 基本类型
  • 引用类型

所有引用类型的值都是Object的实例，数组和对象都是引用类型，修改的只是指针，和基本类型不同。

小结：

  • 基础类型值在内存中占据固定大小的空间，因此被保存在栈内存中；
  • 引用类型的值是对象，保存在堆内存中；
  • 包含引用类型值的变量实际上包含的并不是对象本身，而是一个指向该对象的指针；
  • 从一个变量向另一个变量复制引用类型的值，复制的其实是指针，因此两个变量最终都指向同一个对象。

执行环境

  • 作用
    • 定义了变量或函数有权访问的其他数据
    • 决定了他们各自的行为

小结：

  • 分成全局环境和函数执行环境；
  • 每次进入一个新执行环境，都会创建一个用于搜索变量和函数的作用域链；
  • 全局环境只能访问在全局环境中的定义的变量和函数，而不能直接访问局部变量中的任何数据。
