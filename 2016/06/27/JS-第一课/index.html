JS 第一课

发表于 2016-06-27   |   分类于 前端

语法

  • 区分大小写；
  • 第一个字符必须是字母 || “_” || “$”；
  • 按照惯例，ECMScript 标识符采用驼峰大小写格式，第一个字母小写，色好难过下单词首字母大写；
  • 变量：var 定义的变量成为该变量的作用域中的局部变量，函数退出后销毁；

数据类型

  • 基础数据类型
    - undefine    
- null  空对象指针    
- boolean    
- number     
- string
  • 复杂数据类型
    • object

  • typeof 操作符

    • undefine 未声明或者未初始化

    • boolean 布尔值

      • true

      • false

    • number 数字

      • 十进制（直接输入）
      • 八进制（前面加0）
      • 十六进制（前面加0x）
      • NaN

    • string 字符串

    • object 对象或者null
    • function 函数
  • 数值转换
    • Number()
    • parseInt()
    • parseFloat()
  • 字符串转换
    • toString()
      • null undefined 没有这个方法

操作符

  • 一元操作符（只能操作一个值的操作符）
    • i++
    • ++i
    • i–
    • –i

      递增和递减操作是在包含它们的语句被求值之后才执行。

    • +
    • -

  • 位操作符

    有符号的整数，第三十二位表示数值的正负
    0表示正数，1表示负数
    正数为纯二进制格式存储
    负数要进行三步（补码=反码+1）

    - 绝对值的二进制码
- 反码
- +1
    • NOT(按位非)
      • ~ 返回数值的反码
    • AND（按位与）
      • & （逻辑乘） 两个数值都为1的时候才返回1
    • OR（按位或）
      • | （逻辑加） 两个数值都为0的时候才返回0
    • XOR
      • ^ 两个数值只有一个1时才返回1，否则返回0
    • 左移
      • <<
    • 有符号的右移
      • >>
    • 无符号的右移
      • >>>
      • 正数的无符号右移和正数的有符号右移结果相同
      • 负数的无符号进行补码形式表示
  • 布尔操作符
    • 逻辑非！
    • 逻辑与&&（*）
    • 逻辑或||（+）
  • 乘性操作符
    • 乘法 *
    • 除法 /
    • 求模 %
  • 加性操作符
    • +
    • -
  • 关系操作符
    • > < <= >=
  • 相等操作符
    • ==
    • !=
    • ===

  • 条件操作符

  • 赋值操作符

    • =

  • if 语句

  • do-while 语句

  • while 语句

  • for 语句

  • for-in 语句(枚举对象的属性)

  • break && continue 语句

    • break立即退出循环，强制继续执行循环后面的语句
    • continue 立即退出循环，返回循环顶部

  • switch 语法
