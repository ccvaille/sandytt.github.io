常用插件

    sublime插件控制台，提供添加、删除、禁用、查找插件等功能

    高亮匹配括号、标签

  • Emmet(Zen Coding)

    html/css 快速编辑

常用快捷键

  • 编辑
    • Ctrl + Enter 当前行下面新增一行
    • Ctrl + Shift + Enter 当前行上面增加一行
    • Ctrl + ←/→ 逐词移动
    • Ctrl + Shift + ←/→ 逐词选择
    • Ctrl + Shift + ↑/↓ 移动当前行
    • Ctrl + X 删除当前行
  • 选择
    • Ctrl + D 多重选择
    • F3 选中相同的下一个
    • Shift + F3 选中相同的上一个
    • Ctrl + K + D 跳过选择
    • Ctrl + Shift + L 打散
    • Ctrl + J 合并一行
    • Ctrl + L 选中当前行
    • Alt + F3 选择所有相同的词
  • 查找/替换
    • Ctrl + F
    • Ctrl + H
  • 跳转
    • Ctrl + P 列出当前打开的文件(Go To Anything)
    • Ctrl + R 列出当前文件的符号(类名、函数名)
    • Ctrl + G 跳到指定行数
  • other
    • Ctrl + K, Ctrl + B 显示|隐藏侧栏
    • Ctrl + Shift + N 创建一个新窗口
    • Ctrl + N 新建一个标签
    • Ctrl + W 关闭当前标签
    • Ctrl + Shift + T 恢复刚关闭的标签
    • Alt + Shift + 2 左右分屏
    • Alt + Shift + 8 上下分屏
    • Alt + Shift + 5 上下左右分屏
    • Ctrl + num 跳到指定屏
    • Ctrl + K + K 从光标处删除至行尾
    • Ctrl + K + U 大写
    • Ctrl + K + L 小写
    • Ctrl + K + T 折叠所有属性
    • Ctrl + K + 0 展开所有
    • Ctrl + Shift + [ 折叠代码
    • Ctrl + Shift + ] 展开代码
  • 全屏
    • F11 普通全屏
    • Shift + F11 无干扰全屏
